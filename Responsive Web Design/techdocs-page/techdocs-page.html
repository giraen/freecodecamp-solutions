Getting Started

Before heading out to programming with Arduino, you should first have the following:

  1. Arduino Boards (e.g. Arduino Uno, Arduino Mega, and Arduino Nano)
  2. Arduino IDE
  3. USB Cable
  4. Electronic Components (e.g. resistors, LEDs, and sensors)

In this documentation, we will assume that you uses Arduino Uno for reasons that it is often used by beginners in Arduino. Further, this documentation will only focus on the programming aspect of Arduino. Documentations, for instance, of Arduino boards' schematic diagram and other technical aspects will not be covered but you are welcome to visit Arduino's official website.

For the installation of Arduino IDE, go visit their website.

Getting to know the Basics of Arduino Structure

Upon opening the Arduino IDE, you will be met with the following codes:

void setup() {
// put your setup code here, to run once:

}

void loop() {
// put your main code here, to run repeatedly:

}

From this, we can see two types of function, that is the setup() and loop(). Codes within the setup() will only run once while codes within the loop() will run indefinitely.

Generally, codes within the setup() are for setting up the mode for pins and initialization of some functions such as Serial communication and I2C LCD. This is because it only needs to be executed once. While in loop(), it often contains codes or functions that is necessary to be executed all the time, such as lighting patterns for LEDs, printing for Serial communication or I2C LCD, controlling the motors, and many more.

Data Types and Variables

In arduino programming, we have different types of data. However, we will only focus on the main ones since they are fundamentally necessary for us to learn.

  • int

    Represents whole numbers such as 1, 2, 3 and so on. It is a 16-bit data type, capable of storing values from -32,768 to 32,767.

  • float

    A data type that allows you to store numbers with decimal places, that is for precision. For increased precision, you might want to use double. Take note that it takes more memory for higher precision.

  • byte

    Similar to int, it represents a single byte value, which ranges only from 0 to 255.

  • bool

    Accepts the value of true or false

  • char

    Represents a single character using the ASCII encoding. It can store a single character like 'a', '1', etc.

  • String

    Similar to char but it can work with string of characters, that is words, senteces, or phrases.

For declaring and initialization of variables in arduino, we follow this syntax.

data_type variable_name = value;

Take note that you cannot have a variable name that has spaces within it. A good example is of declaring and initializing a variable is like this.

int countUp = 0;

You can also just declare the variable above with no value like this.

int countUp;
Printing Hello World

There are many ways to print hello world in arduino, the classic one is using the Serial. If you explored your IDE, you should be able to see the Serial Monitor in the upper right.

To start, we will initialize our Serial with the begin function. Make sure that it is placed within the squiggly brackets of setup().

Serial.begin(9600);

This line of code follows this syntax.

Serial.begin(baudrate);

Baudrate refers to the data transmission of your arduino to your laptop or desktop computer. 9600 is the common baudrate. Increasing the baudrate will result to faster transmission while decreasing it will result to a slower transmission. Take note that it should match the baudrate in code and in the serial monitor because mismatch in baudrate between the code and serial monitor will result to random gibberish of random characters.

After that, we can use the print and/or println function inside the squiggly brackets of loop().

Serial.print("Hello world"); Serial.println("Hello world");

The difference between the two is that the print() will always print whatever inside of it within the same line. Therefore Serial.print("Hello world"); is also the same as Serial.print("Hello"); Serial.print(" "); Serial.print("World");

On the other hand, println() will print whatever inside of it to a new line.

Take note that if it is a string or a character, as long as it contains letters, it must be always within the double quotation marks.
